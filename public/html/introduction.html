成人英语课程

自创的庞大学习库，囊括了工作与生活的方方面面，让你在任何情况下，都可以应对自如。

  • 成人口语

  • 适用人群：
    想提高口语的学习者
  • 适用级别：
    L1-L12
  • 课程介绍：
    16大主题，全天欧美外教，致力于口语综合能力提升。

  • 零基础

  • 适用人群：
    基础薄弱人士，无法开口表达，寻求口语突破学习者
  • 适用级别：
    L0
  • 课程介绍：
    见词能读，听音能写，基础技巧口语课程，所有课程1v1名师专属指导。

  • 蹭课卡

  • 适用人群：
    无固定上课时间，利用空余时间提高口语的学习者
  • 适用级别：
    L1-L12
  • 课程介绍：
    16大主题，全天欧美外教，致力于口语综合能力提升。

  • 旅游英语

  • 适用人群：
    短期内有出国旅游需求的学习者
  • 适用级别：
    L0-L12
  • 课程介绍：
    学习地道的语句表达和对话练习，补充丰富的基础词汇，独自完成国外旅行不是梦。

  • 面试英语

  • 适用人群：
    职场人士，短期内有工作面试需求的学习者
  • 适用级别：
    L0-L12
  • 课程介绍：
    提升学员制作英文简历以及参与英语面试的综合能力。

  • 自然拼读

  • 适用人群：
    学习的基本发音，快速提高口语者
  • 适用级别：
    L0-L12
  • 课程介绍：
    掌握所有发音规律，学完就能见词会读，听音能写
